DUO SDK Compilation
------------------------------------
Samples Overview - C/C++ (Qt)
------------------------------------

This project demonstrate simple usage of DUO3D with OpenCV and Qt.

------------------------------------
Compilation Instructions
------------------------------------

Requirements:

	- Visual Studio 2013 (Windows)
	- GCC (Linux)
	- XCODE (Mac)
	- OpenCV v2.4.10
	- Qt v5.1.1+
	- QtCreator
	
	------------------------------------
	Building Samples
	------------------------------------

	All Platforms: 

	1) Open Sample.pro file in QtCreator
	2) Build the sample
	3) The executable will be placed in bin folder
	3) Plug in your DUO Device and run the sample
